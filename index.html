Web API for Hacker News

Service that will allow users to access the content hosted on Hacker News. The content is delivered in non-html representation. On JSON support at present (no plans to change that). There is no persistence of the content in (aside from possible caching).

The purpose of this API is to take some of the pain involved with developing toy applications that use the Hacker News data. I mean, who really likes parsing html?
